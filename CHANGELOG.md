# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.7] - 2026-05-04

### Added
- Initial release of 微光日记 (Mood Note)
- Three-tab navigation: Calendar, Record, Profile
- Emoji mood picker with 9 mood types
- Daily record functionality (what happened, happy events, unhappy events)
- Photo upload and cover photo selection
- Life tags system
- SQLite local database
- Calendar view with mood emoji display
- Day detail page
- Settings page with theme and AI settings
- Time capsule feature (basic)

### Technical
- React Native 0.76.9 with Expo 52
- TypeScript support
- Expo Router for navigation
- Expo SQLite for local storage

## [0.1.0] - 2026-04-01

### Added
- Project initialization
- Basic Expo setup
